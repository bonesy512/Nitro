# Next.js, Material UI, Styled Components and Typescript Example

![NMST](screenshot.png)

## NPM

```bash
git clone https://github.com/williamluke4/nextjs-materialui-styledcomponents.git
cd nextjs-materialui-styledcomponents

npm install
npm run dev
```

## Yarn

```bash
git clone https://github.com/williamluke4/nextjs-materialui-styledcomponents.git
cd nextjs-materialui-styledcomponents

yarn
yarn dev
```
